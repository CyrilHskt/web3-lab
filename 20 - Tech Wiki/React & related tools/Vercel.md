Vercel = plateforme cloud orientée frontend pour build / déployer / scaler des sites et apps web modernes (surtout Next.js, React, Vue, etc.).​
- Déploiement Git‑based quasi instantané (push → build → déploy global) avec previews immuables pour chaque branche / PR.​
- Infra managée : CDN global + edge network, serverless / edge functions, auto‑scaling, SSL automatique, custom domains, env vars.​
- Nativement optimisé pour Next.js (ISR, SSR, edge functions, image optimization, zero‑config), c’est la “plateforme maison” de Next.​
- Objectif : enlever toute la gestion d’infra et ops pour que tu te concentres sur le frontend / produit, tout en gardant perf, scalabilité et DX au top.​
